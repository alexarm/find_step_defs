# find_step_defs

Before use:
```
pip install progress
```

Usage:
```
usage: find_step_defs.py [-h] [-a STEPDEFS_DIRECTORY]
                         [-f STEPDEFS_DIRECTORY FEATURES_DIRECTORY]
                         [-u STEPDEFS_DIRECTORY FEATURES_DIRECTORY]

optional arguments:
  -h, --help            show this help message and exit
  -a STEPDEFS_DIRECTORY, --all STEPDEFS_DIRECTORY
                        Find all steps
  -f STEPDEFS_DIRECTORY FEATURES_DIRECTORY, --features STEPDEFS_DIRECTORY FEATURES_DIRECTORY
                        Find all steps with features
  -u STEPDEFS_DIRECTORY FEATURES_DIRECTORY, --unused STEPDEFS_DIRECTORY FEATURES_DIRECTORY
                        Find all unused steps
```
